# Восьмая практическая работа в Я.Практикуме: Сервис Mesto.

**Описание проекта**

[Ссылка на проект](https://leonid-dolgodvorov.github.io/mesto/index.html)

* HTML;

* Адаптивная вёрстка;
* js - popup, вставка шаблона через template из массива, верификация форм, классы.
* сборка webpack-ом
